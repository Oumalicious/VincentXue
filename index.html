My proudest project is a vocabulary exam maker, which utilizes DOM manipulation and JSON files. There are 3 aspects when it comes to learning character based languages: the character itself (樂/乐), pronunciation (Lè), and the translation (happy). Initially, this was a whimsical project that helped me study for a Chinese test, but I saw its potential to be dynamic and universal. Tests are editable and combinable, can save incorrect answers into a test, and logs the elapsed time per each question. I showed the app to my friend who was pursuing a Master's in Occupational Therapy, and it provided positive results. I see a future where anybody can use my app to study.

Character Reviewer App (demo shown in the Coding)
Link: https://oumalicious.github.io/VocabularyReviewer/
  • Uses a double multiple-choice format, testing the character's pronounciation and translation to the native tongue.
  • Allows the user to create their own tests by saving it into a JSON. Tests can be combined and editted.
  • Features a question timer for rapid testing or progress checking, and a test timer to simulate a timed environment.
  • The results display the user's choices, answer key, and the questions that were marked incorrect, which can be saved into a test/JSON.
